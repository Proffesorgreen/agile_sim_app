# Agile Project Management Plan
## AASTU Library Management System
## 1. Project Overview
The AASTU Library Management System allows students to register, borrow books,
return them, and view their borrowing history. Librarians can manage the
catalogue,
monitor borrowings, and access usage analytics. This plan describes how the system
is built using the Scrum Agile framework.
